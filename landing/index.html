Your own web IDE.
Self-hosted. Single binary.

A lightweight, open-source code editor, terminal, and git client that runs in your browser. No cloud, no subscriptions. Run it on your server, NAS, or Raspberry Pi.

Get Started Free Star on GitHub

Install in seconds

Wede is a single binary written in Go. No dependencies, no complex setup. Just run the install script and you're ready to code.

  • Zero dependencies (no Node, no Docker)
  • Lightweight (~10MB binary)
  • Works on Linux, macOS, and ARM (Raspberry Pi)
Install Script
$ curl -fsSL "http://wede.pty.it.com/install.sh" | bash
Quick Start
# 1. Start wede
$ wede .
# 2. Open in browser
$ open http://localhost:9090

Pro-grade Code Editor

wede provides a fast, responsive editing experience with syntax highlighting for 20+ languages including Go, Rust, Python, and TypeScript.

Multiple Themes

Midnight and Daylight themes to suit your environment.

File Explorer

Full project tree with git status and file management.

Visual Git Control

Manage your source control without leaving the browser. Stage changes, commit, and visualize your branch history with a built-in commit graph.

  • Interactive commit graph
  • Staging and Diff views
  • Branch management

Terminal & Browser

A full PTY terminal emulator and an embedded browser preview let you run commands and see results side-by-side. No more context switching.

WebSocket Powered

Real-time terminal response with full support for interactive CLI tools.

Live Preview

Embedded browser tab for instant feedback on your web applications.

Everything you need, nothing you don't

Designed for speed, security, and simplicity.

Mobile Ready

Fully responsive UI that works beautifully on iPads and tablets. Code from the couch or on the go.

Secure by Design

Password protection with brute-force lockout. Deploy behind your own VPN or reverse proxy for maximum privacy.

Highly Configurable

Customizable ports, working directories, and themes via a simple JSON configuration file.

Hack on wede

Wede is open-source and built for developers. The frontend is React + Vite, and the backend is pure Go.

Frontend Dev

$ npm install
$ npm run dev

Backend Dev

$ cd backend
$ go run ./cmd/wede .
