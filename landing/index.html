Get started in seconds

One command. That's it.

Shell
$ curl -fsSL https://raw.githubusercontent.com/webcrft/wede/main/install.sh | bash

Or download the binary directly from GitHub Releases.

Up and running

Create a config, start wede, open your browser.

1. Create a config file
$ cat wede.config.json { "password": "your-password", "port": "9090" }
2. Start wede
$ wede /path/to/your/project
3. Open your browser
$ open http://localhost:9090

Development

Want to hack on wede itself? Run the frontend and backend separately.

Frontend

React + Vite dev server with hot reload.

$ npm install $ npm run dev

Backend

Go server with file system, terminal, and git APIs.

$ cd backend $ go run ./cmd/wede .
Build a single binary
$ npm run build:all

Why wede?

A self-hosted alternative to cloud IDEs like GitHub Codespaces, Gitpod, and code-server.

Zero dependencies

One ~10MB binary. No Docker, no Node.js runtime, no database. Download it, add a config file, run it. Works on any Linux server, macOS workstation, or ARM device like Raspberry Pi.

Your data stays yours

No cloud accounts, no telemetry, no vendor lock-in. Your source code never leaves your machine. Perfect for air-gapped environments, private networks, and compliance-sensitive projects.

Built for remote development

Access your full development environment from any device with a browser. Code on your laptop, continue on your tablet, or pair-program through a shared URL. Works over SSH tunnels and VPNs.
